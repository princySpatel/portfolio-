My 2D Platformer

I made a 2D platformer game using the Godot Engine, drawing inspiration from classic titles like Super Mario Bros. I designed and created all the graphics and assets for the game myself, including the player character, enemies, and level environments. I wrote all the game's code in GDScript, implementing core mechanics like player movement, collision detection, and enemy AI. This project demonstrates my skills in game development, from asset creation to scripting and overall game design.

Java Snake Game

This project is a complete implementation of the classic Snake game, built from the ground up in Java. The game showcases my ability to develop a fully functional application, with a focus on core programming concepts and game logic.

Key Features:

Core Game Loop: I implemented the fundamental game mechanics, including snake movement, food generation, and a robust collision detection system.

Score Tracking: The game includes a basic but clear visual interface that displays the player's current score, providing a sense of progression.

Game State Management: I developed a functional "Game Over" screen that is triggered when the snake collides with itself or the walls, providing a definitive end to the game.

Unity Terrain and Robot Movement Demo

This project showcases a fundamental tech demo built in Unity, serving as a versatile foundation for various game genres. It features a custom-built terrain and integrates Unity's default robot assets. The core functionality includes:

Custom Terrain Generation: A detailed, procedural terrain designed to demonstrate environmental design and navigation.

Default Robot Integration: Utilizes Unity's built-in robots to showcase basic character control and movement scripting.

Executable Build: The project is compiled into a standalone executable (.exe) for direct download and play on Windows-based PCs, highlighting the ability to create and deploy a finished product.
